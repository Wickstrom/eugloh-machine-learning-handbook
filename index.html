Machine Learning with Python — Lecture Slides

EUGLOH course "Problem Solving Using Open-Source Languages; R and Python" — University of Novi Sad

Plain notebooks (.ipynb) are rendered with jupyter nbconvert --to slides, the same Reveal.js pipeline that RISE uses, so the nested horizontal/vertical slide structure matches what you would see live in RISE. They are static HTML served from GitHub Pages, so they never time out.
Marimo notebooks (.py) are exported with marimo export html-wasm and run entirely in your browser via WebAssembly — the Python interpreter, dependencies, and code all live inside the page. Reactive widgets (sliders, dropdowns, tabs) update figures live. Give them a few seconds to spin up on first load.
